## ✨ Features
- 🎮 Play Rock–Paper–Scissors against the computer  
- 🧠 Computer generates random choices  
- 🏆 Instant result display (Win / Lose / Draw)  
- 🔄 Option to play unlimited rounds  
- 🖥️ Works on computers and laptops
